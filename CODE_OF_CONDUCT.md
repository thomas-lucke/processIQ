# Code of Conduct

This project is committed to a respectful, harassment-free environment for contributors, reviewers, and users.

## Expected Behavior

Examples of behavior that help the project:

- being respectful in technical disagreement
- giving constructive, evidence-based feedback
- assuming good intent while still being clear about risks and trade-offs
- keeping discussion focused on the work and its users

## Unacceptable Behavior

Examples of unacceptable behavior include:

- personal attacks, insults, or degrading language
- harassment, discrimination, or intimidation
- publishing someone else's private information without permission
- sustained disruption of issues, pull requests, or review threads

## Scope

This code of conduct applies to all project spaces, including:

- GitHub issues
- pull requests and review comments
- discussions and related community channels
